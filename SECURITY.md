# Security Policy

## Supported Versions

Security fixes are provided for the latest version of Crystal Chat.

## Reporting a Vulnerability

If you believe you’ve found a security or privacy vulnerability, please report it privately.

- Email: help.crystalgames@gmail.com
- Subject: [SECURITY] Crystal Chat vulnerability report

Please include:

- A clear description of the issue and impact
- Steps to reproduce (proof-of-concept if available)
- Affected version / commit
- Your OS and any relevant environment details

## Disclosure Process

After receiving your report:

- We’ll acknowledge receipt within a reasonable timeframe.
- We’ll investigate and work on a fix.
- If needed, we may ask follow-up questions to validate the report.
- Once a fix is available, we’ll coordinate disclosure details.

Thank you for helping keep Crystal Chat secure.
